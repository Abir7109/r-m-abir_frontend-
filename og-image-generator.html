  1. Open this file in Chrome/Edge browser (for best quality)
  2. Press F11 to enter fullscreen mode
  3. Take a screenshot of just the blue card (1200x630px):
    • Windows: Use Snipping Tool or Win + Shift + S
    • Mac: Use Cmd + Shift + 4
  4. Save the screenshot as og-image.jpg
  5. Place it in assets/img/og-image.jpg
  6. Commit and push to deploy!

Note: The image MUST be exactly 1200x630 pixels for best results on all platforms.

Alternative: Use a screenshot tool with fixed dimensions like Awesome Screenshot (Chrome extension) or ShareX (Windows).
